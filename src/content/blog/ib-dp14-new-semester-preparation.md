こんにちは。IB専門家庭教師サービス「IBT（IB Tutors）」です。

もうすぐ4月。IB DP（ディプロマ・プログラム）の1年目がスタートする時期です。

「DPが始まるけど、何を準備すればいいの？」「MYPからDPに上がるけど、レベルが上がるって聞いて不安…」

そんなIB DP1年生（とその保護者の方）に向けて、**4月の新学期までにやっておくべき準備**をIB卒業生の視点からお伝えします。

**結論：DPは"始まってから頑張る"では遅い。事前準備が2年間の成績を左右する。**

---

## なぜ事前準備が重要なのか

IB DPが始まると、授業のスピードと課題の量に圧倒されます。

### MYPとDPの違い

| | MYP | DP |
|--|-----|-----|
| **科目数** | 8教科群 | 6科目 + TOK + EE + CAS |
| **評価方式** | 1-7（学校内評価中心） | 1-7（外部試験 + 内部評価） |
| **課題の量** | 多いが管理可能 | 非常に多い。IA、EE、TOKが追加 |
| **授業のスピード** | 比較的ゆっくり | 2年間で全範囲を終わらせる必要あり |
| **自主性の求められ度** | 教師のガイダンスが多い | 自分で計画・実行する力が必須 |

DPの1年目は**全科目のIA（内部評価）が本格的にスタート**し、EEのテーマ探しも始まります。

「まだ始まったばかりだし、ゆっくりやろう」と思っていると、**気づいたときには手遅れ**になるのがDPの怖さです。

## 準備1: 選択科目のシラバスを読む

DP科目のシラバス（学習要項）は、IBOの公式サイトや学校から配布されます。

### やるべきこと

- **各科目のシラバス概要を確認する**：どんなトピックを学ぶのかを把握する
- **HL科目は特に注意**：SLとの違い（追加トピック、追加の学習時間）を理解する
- **IAの要件を確認する**：各科目でどんな内部評価が課されるのかを事前に知っておく

### 特に重要な科目

| 科目 | 事前に確認すべきこと |
|------|-------------------|
| **Math AA/AI** | 扱う単元のリスト。特にHLは微積分・統計の範囲が広い |
| **Sciences（Physics/Chemistry/Biology）** | 実験（ラボ）の内容とIAのテーマ選びの基準 |
| **English A / Japanese A** | 読む文学作品のリスト。先に読み始められるとアドバンテージ |
| **History / Economics** | 扱う時代・テーマのリスト。予備知識があると授業理解が速い |

## 準備2: 文学作品を先読みする

Group 1（言語と文学）とGroup 2（言語習得）では、**文学作品の精読とエッセイ執筆**が中心です。

### やるべきこと

- **学校から指定された文学作品リストを確認する**
- **少なくとも1〜2冊は読み始める**
- **読みながらメモを取る習慣をつける**：テーマ、登場人物、象徴的な表現など

### 文学作品の読み方のコツ

1. **1回目は通読**：ストーリーの流れを掴む
2. **2回目は精読**：気になった表現やテーマをメモしながら読む
3. **「なぜこの表現を使ったのか？」を常に考える**：これがIBの文学分析の基礎

**DPが始まってから作品を読む時間を確保するのは非常に難しい**です。春休みのうちに1〜2冊読んでおくだけで、授業のスタートダッシュが全然違います。

## 準備3: 数学の基礎を固める

DPの数学（Math AA / Math AI）は、MYPの数学から**一気にレベルが上がる**科目です。

### Math AA を選んだ場合

以下の基礎が確実にできているか確認してください。

| 分野 | 確認ポイント |
|------|-----------|
| **代数（Algebra）** | 因数分解、方程式の解法、不等式 |
| **関数（Functions）** | 一次関数、二次関数、グラフの変換 |
| **三角法（Trigonometry）** | sin, cos, tanの基本、三角方程式 |
| **指数・対数** | 指数法則、対数の基本性質 |

HLの場合は、**数学的帰納法や複素数**の予習もしておくと有利です。

### Math AI を選んだ場合

| 分野 | 確認ポイント |
|------|-----------|
| **統計（Statistics）** | 平均、中央値、標準偏差の計算方法 |
| **確率（Probability）** | 基本的な確率の計算、条件付き確率 |
| **グラフ読解** | データからグラフを読み取り、傾向を説明する力 |
| **GDC（グラフ電卓）の操作** | TI-84やCasio ClassPadの基本操作 |

### GDC（グラフ電卓）に慣れておく

IB DPの数学では、**GDC（Graphic Display Calculator）**の使用が必須です。

- 学校指定の電卓（TI-84 Plus CE、TI-Nspire、Casio fx-CG50 など）を入手する
- 基本操作（グラフ描画、方程式の解法、統計計算）を練習する
- **試験でGDCを使いこなせるかどうかが、得点に直結する**

春休みのうちに電卓を手に入れ、**YouTubeのチュートリアル動画で基本操作を覚える**のがおすすめです。

## 準備4: エッセイライティングの基礎を身につける

DPでは、**ほぼすべての科目でエッセイ（論文形式の文章）を書く力**が求められます。

### IB式エッセイの基本構造

```
Introduction（序論）
  - テーマの提示
  - 自分の主張（thesis statement）

Body Paragraph 1（本論1）
  - 主張を支える根拠1 + 具体例

Body Paragraph 2（本論2）
  - 主張を支える根拠2 + 具体例

Body Paragraph 3（反論への対応）
  - 予想される反論 + それに対する自分の考え

Conclusion（結論）
  - 主張の再確認 + テーマの広がり
```

### 練習方法

1. **好きなテーマで500語のエッセイを書いてみる**
   - 例：「SNSは学生にとって有益か？」「制服は必要か？」
2. **必ず「反論」を入れる練習をする**
   - IBでは、自分の意見だけでなく**対立する視点を検討する力**が評価される
3. **書いたら誰かに読んでもらう**
   - 保護者でも友人でもOK。「論理が通っているか」「根拠があるか」をチェックしてもらう

## 準備5: CASの計画を始める

CAS（Creativity, Activity, Service）は、18ヶ月間の継続が求められるIB DPの必修要素です。

### 4月までにやるべきこと

1. **CASの3要素を理解する**
   - **Creativity**：創造的な活動（音楽、美術、プログラミング、料理など）
   - **Activity**：身体的な活動（スポーツ、ダンス、ハイキングなど）
   - **Service**：社会貢献活動（ボランティア、地域活動、環境保全など）

2. **すでにやっている活動をリストアップする**
   - 部活動、習い事、ボランティアなど、CASに該当するものを整理する
   - **新しく始める必要はない**。既存の活動をCASに組み込めることが多い

3. **CASプロジェクトのアイデアを考え始める**
   - CASプロジェクトは「計画 → 実行 → 振り返り」を含む活動
   - 例：「地域の子どもたちにプログラミングを教えるワークショップを企画する」
   - 学校の文化祭の企画をCASプロジェクトにするのも一般的

## 準備6: 時間管理のツールを用意する

DPの2年間を乗り切るために、**時間管理の仕組みを4月の前に整えておく**ことを強くおすすめします。

### おすすめのツール

| ツール | 用途 | 特徴 |
|--------|------|------|
| **Googleカレンダー** | 締め切り管理 | リマインダー機能が便利。色分けで科目別管理 |
| **Notion / OneNote** | ノート・タスク管理 | 科目別にノートを整理。チェックリスト機能あり |
| **タイマーアプリ** | 集中時間の管理 | ポモドーロ・テクニック（25分集中＋5分休憩）に使う |

### 4月までにセットアップすること

1. **カレンダーに年間の大まかなスケジュールを入れる**（期末試験、IA提出日など）
2. **科目別のフォルダ/ノートを作る**
3. **毎週日曜日に「翌週の予定確認」の時間を15分ブロックする**

## 準備7: 心と体のコンディションを整える

技術的な準備も大事ですが、**心身のコンディション**も同じくらい重要です。

### 春休みにやっておきたいこと

- **十分な睡眠をとる習慣をつける**：DPが始まると睡眠時間が削られがち。今のうちに「早寝早起き」のリズムを作る
- **運動の習慣を維持する**：DPの勉強漬けになると運動が減る。CASのActivityにもなるので、続けられる運動を決めておく
- **趣味や息抜きの時間を大切にする**：DPが始まっても「これがあるから頑張れる」という楽しみを持っておく
- **友人関係を大切にする**：同じDPを乗り越える仲間がいることは、想像以上に心の支えになる

## 保護者の方へ -- 新学期に向けてのアドバイス

### やっておくと良いこと

1. **IBコーディネーターとの連絡手段を確認する**：何かあったときにすぐ相談できるように
2. **DPの2年間のスケジュール感を把握する**：いつ頃が忙しくなるか、締め切りが集中するのはいつかを知っておく
3. **お子さまの「頑張りすぎ」を見守る**：真面目なIB生ほど、限界を超えて頑張ってしまう傾向がある
4. **家庭での学習環境を整える**：静かな勉強スペース、必要なデバイスやWi-Fi環境

### 避けてほしいこと

- **「DPは大変だから覚悟して」と脅さない**：不安を煽るのではなく、「一緒にサポートするよ」という姿勢で
- **他のIB生と比較しない**：科目も学校も違うため、比較に意味がない
- **過度に干渉しない**：DPでは自己管理能力を養うことも大切。見守りつつ、求められたときにサポートする
