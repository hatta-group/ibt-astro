「IBの授業についていけない」「DPの試験対策を誰かに教えてほしい」「IB専門の家庭教師ってどこがいいの？」

IB（国際バカロレア）は独自のカリキュラムで進むため、一般的な学習塾や家庭教師では対応が難しいのが現実です。だからこそ、**IB専門の家庭教師・塾選び**が成績アップのカギになります。

この記事では、**IB卒業生の講師陣が実際に調査した情報**をもとに、日本国内のIB家庭教師・塾サービス7社を料金・講師・対応プログラムの面から徹底比較します。

「どこを選べばいいのかわからない」という保護者の方、「自分に合ったサポートを見つけたい」というIB生の皆さんに、最適なサービスが見つかるようお手伝いします。

## IB家庭教師の選び方｜5つのポイント

IB専門の家庭教師・塾を選ぶ際に、必ず確認しておきたい5つのポイントをまとめました。

### 1. 講師がIB経験者かどうか

IBカリキュラムは一般的な学校教育とは大きく異なります。TOK（Theory of Knowledge）やEE（Extended Essay）、IA（Internal Assessment）といったIB独自の課題に対応できるのは、**実際にIBを経験した講師**です。

「帰国子女」や「英語が得意」というだけでは、IBの学習指導には不十分な場合があります。講師がIB卒業生かどうか、IBスコアをどの程度取得しているかを確認しましょう。

### 2. 料金体系（月額制 vs 都度払い）

IB家庭教師・塾の料金体系は主に2種類あります。

- **月額制:** 毎月一定額を支払う方式。安定した学習スケジュールが組めるが、途中でやめにくい
- **都度払い:** 受講した分だけ支払う方式。気軽に始められ、必要な時だけ利用できる

入会金・管理費・年会費なども含めた**トータルコスト**で比較することが大切です。

### 3. 対応プログラム（PYP / MYP / DP）

IBには3つのプログラムがあります。

- **PYP（Primary Years Programme）:** 3歳〜12歳
- **MYP（Middle Years Programme）:** 11歳〜16歳
- **DP（Diploma Programme）:** 16歳〜19歳

サービスによってはDP（高校）しか対応していない場合もあります。お子さんのプログラムに対応しているか、事前に確認してください。

### 4. 無料体験の有無

IB家庭教師は講師との相性が重要です。実際に授業を受けてみないとわからないことも多いので、**無料体験授業があるサービス**を優先的に検討するのがおすすめです。

体験授業の長さも確認しましょう。15分の短い面談だけでは、授業のイメージがつかめません。

### 5. 実績・社会的証明

合格実績やスコアアップの事例、保護者・生徒の体験談が公開されているかどうかも重要な判断材料です。

ただし、実績の見せ方はサービスによって異なります。講師のプロフィールが詳しく公開されているか（実名・出身校・IBスコアなど）も、信頼性を測る目安になります。

## IB家庭教師・塾 おすすめ7社比較

日本国内で利用できるIB対応の家庭教師・塾サービスを7社ピックアップし、それぞれの特徴を紹介します。

### 1. IBT（IB Tutors）

**IB完全特化のオンライン家庭教師。全講師がIB卒業生。**

| 項目 | 内容 |
|------|------|
| 対応プログラム | PYP / MYP / DP |
| 講師 | 6名（全員IB卒業生） |
| 料金 | PYP 5,808円 / MYP 7,260円 / DP 8,712円（60分あたり） |
| 入会金 | **0円** |
| 管理費 | **なし** |
| 契約形態 | **都度払い** |
| 無料体験 | **60分無料** |
| 形式 | オンライン |

**特徴:**

IBTは事業の100%がIBに特化しているサービスです。講師は6名と少数精鋭ですが、**全員がIB卒業生**で、講師のプロフィール（実名・出身校・IBスコア）がサイト上で公開されています。

最大の特徴は**入会金0円・管理費なし・都度払い**という料金体系。月額制のように「毎月固定費がかかる」ということがなく、必要な時に必要な分だけ受講できます。

さらに、IB校データベース（162校分の個別ページ）やIBerタイプ診断など、IBに特化した情報コンテンツも充実しています。この記事を書いているのも、IBTのIB卒業生講師陣です。

**メリット:**
- 入会金0円、管理費なし、都度払いで始めやすい
- 全講師がIB卒業生（プロフィールを全員公開）
- 60分の無料体験授業がある
- PYP/MYP/DPの全プログラムに対応
- 料金がサイト上で明確に公開されている

**デメリット:**
- 講師数が6名と少ない
- オンライン専用（対面指導なし）
- サービス開始が2024年と比較的新しい

### 2. EDUBAL

**帰国子女向け総合オンライン家庭教師。講師数は業界最大級。**

| 項目 | 内容 |
|------|------|
| 対応プログラム | PYP / MYP / DP |
| 講師 | 約4,000名（全員帰国子女） |
| 料金 | 7,500円/60分 |
| 入会金 | 22,000円 |
| 管理費 | 3,300円/月 |
| 契約形態 | 月謝制 |
| 無料体験 | 初回無料（通常2,420円） |
| 形式 | オンライン |

**特徴:**

株式会社トモノカイが運営する帰国子女向けの総合オンライン家庭教師サービスです。IBは16コースあるうちの1つで、IB以外にもSAT・TOEFL・帰国子女受験などに対応しています。

講師数は約4,000名と圧倒的な規模を誇りますが、全員が「帰国子女」であり、**IB卒業生かどうかは保証されていない**点は注意が必要です。

合格体験談が大学名付きで公開されており（東大・メルボルン大など）、社会的証明が強いサービスです。

**メリット:**
- 講師数が約4,000名と業界最大級
- 合格実績を大学名付きで公開
- 歴史と知名度がある

**デメリット:**
- IB専門ではなく16コースの1つ
- 講師がIB卒業生かどうかは不明確
- 入会金22,000円+管理費3,300円/月が別途必要

### 3. IBアカデミー

**IB専門のオンライン塾。月額制でカリキュラム型のサポート。**

| 項目 | 内容 |
|------|------|
| 対応プログラム | PYP / MYP / DP / 都立国際IB受験 / IB認定校受験 |
| 講師 | 「150名のIB卒業生+10名のプロ講師」と公式サイトに記載（講師紹介ページには10名をイニシャルで掲載） |
| 料金 | DPビギナープラン: 月額49,800円（税込）〜。他プランは要カウンセリング |
| 入会金 | 非公開 |
| 管理費 | 非公開 |
| 契約形態 | 月額制 |
| 無料体験 | Google Meetで1時間の無料相談 |
| 形式 | オンライン |

**特徴:**

2024年6月設立、2026年3月に公式サイトをリニューアル公開したIB専門のオンライン塾です。IBTと同じくIBに完全特化しており、5コース（DP・MYP・PYP・都立国際IB受験・IB認定校受験）を展開しています。

月額制のカリキュラム型で、「専用チャット365日進捗管理」「月次報告書」など手厚いサポートを打ち出しています。

講師紹介ページには10名の講師がイニシャル表記で掲載されています。公開料金はDPビギナープランの月額49,800円(税込)〜のみで、その他のプラン・コースの料金はカウンセリングで個別に提示される形式です。

**メリット:**
- IB完全特化で5コースの細かい対応
- 都立国際IB受験・IB認定校受験の専門コースがある
- LINE相談で気軽に問い合わせ可能

**デメリット:**
- 月額49,800円〜と料金が高め
- 料金の大部分が非公開（カウンセリング必須）
- 講師紹介がイニシャル表記で詳細が見えにくい
- 2026年3月公開のため実績がまだ少ない

### 4. TCK Workshop

**帰国子女向け総合オンライン家庭教師。プロ講師から学生講師まで幅広いラインナップ。**

| 項目 | 内容 |
|------|------|
| 対応プログラム | PYP / MYP / DP |
| 講師 | プロ講師+学生講師（ランク制。講師数は非公開） |
| 料金 | 講師ランクで変動（プレミアム講師: 18,000円/60分） |
| 入会金 | 22,000円 |
| 管理費 | 4,400円/月 |
| 契約形態 | 月謝制 |
| 無料体験 | 非公開 |
| 形式 | オンライン |

**特徴:**

海外子女・帰国子女向けの総合オンライン家庭教師サービスです。IBだけでなく、帰国子女受験やインター受験にも対応しています。世界55カ国で1,750名以上の指導実績があります。

講師にはランク制が導入されており、プロ講師は質が高い反面、料金も高額です。

**メリット:**
- 世界55カ国・1,750名以上の指導実績
- プロ講師の存在（学生講師だけではない）
- 30分から受講可能で柔軟

**デメリット:**
- IB特化ではない
- プレミアム講師は18,000円/60分と高額
- 入会金22,000円+管理費4,400円/月が別途必要

### 5. オンライン家庭教師WAM IBコース

**総合学習塾WAMのIB専門コース。現職のIB教員も在籍。**

| 項目 | 内容 |
|------|------|
| 対応プログラム | PYP / MYP / DP（G1-G12） |
| 講師 | 13名以上（IB認定校の現職/元教員、IBDP取得者など） |
| 料金 | 非公開（問い合わせ必須） |
| 入会金 | 11,000円程度 |
| 管理費 | 3,200円/月+年会費11,000円程度 |
| 契約形態 | 月謝制 |
| 無料体験 | あり |
| 形式 | オンライン |

**特徴:**

総合学習塾WAMが展開するIB専門コースです。講師にIB認定校の現職教員がいることが他社にない特徴で、日本人学習アドバイザーとの二人体制でサポートする仕組みがあります。

実名・顔写真付きの成功事例（19点から35点へのスコアアップなど）が掲載されており、説得力があります。

**メリット:**
- IB認定校の現職教員が講師にいる
- 日本人学習アドバイザーとの二人体制
- 無料体験授業がある
- フリーダイヤルで電話相談可能

**デメリット:**
- 総合塾のIBコースのため専門の深さに限りがある
- 料金が非公開
- IB専用のコンテンツが少ない

### 6. Kokusaba

**インターナショナルスクール生向けの個別指導。対面レッスンにも対応。**

| 項目 | 内容 |
|------|------|
| 対応プログラム | IB / AP / SAT 等 |
| 講師 | 東大・慶應・Cornell・UCLA等の出身（講師数非公開） |
| 料金 | 非公開 |
| 入会金 | 非公開 |
| 管理費 | 非公開 |
| 契約形態 | 非公開 |
| 無料体験 | 1時間無料 |
| 形式 | オンライン+対面+カフェレッスン |

**特徴:**

インターナショナルスクール生を対象とした個別指導サービスです。IB以外にもAP・SAT・IGCSEなどに対応しています。

「学生による学生のための」というコンセプトで、親しみやすいブランディングが特徴。オンラインだけでなく対面やカフェでのレッスンにも対応しているのは他社にないポイントです。

**メリット:**
- オンライン+対面+カフェの3形態で受講できる
- 1時間の無料体験がある
- モダンなサイトデザインで情報が見やすい

**デメリット:**
- IB特化ではない（AP・SAT等と並列）
- 料金が非公開
- 講師数やIB卒業の有無が不明

### 7. EDUBAL Academy

**EDUBAL系列のIB専門塾。試験直前の集中対策に強み。**

| 項目 | 内容 |
|------|------|
| 対応プログラム | Pre-DP（G9-10）/ IBDP（G11-12） |
| 講師 | EDUBALのIB経験教師 |
| 料金 | 試験直前対策: 90分x6回 135,000円（税込） |
| 入会金 | 非公開 |
| 管理費 | 非公開 |
| 契約形態 | コース制 |
| 無料体験 | 非公開 |
| 形式 | オンライン |

**特徴:**

EDUBAL（トモノカイ）が別ブランドで展開するIB専門の塾サービスです。個別指導ではなく「塾」形式で、試験直前の集中対策コースなどを提供しています。

EDUBALのブランド力がありますが、サイト上の情報量が少なく、料金の詳細も見えにくい状態です。

**メリット:**
- 大手EDUBALのブランド力と信頼性
- 試験直前の集中対策コースがある

**デメリット:**
- PYP非対応（Pre-DP以上のみ）
- 試験対策だけで135,000円と高額
- サイトの情報量が少ない
- 無料体験の有無が不明

## IB家庭教師・塾 料金比較表

各社の料金・条件を一覧で比較します。

| サービス | 入会金 | 授業料（60分換算） | 月額管理費 | 無料体験 | 契約形態 | 講師のIB経験 |
|---------|--------|-------------------|-----------|---------|---------|------------|
| **IBT** | **0円** | PYP 5,808円 / MYP 7,260円 / DP 8,712円 | **なし** | **60分無料** | **都度払い** | **全員IB卒業生** |
| EDUBAL | 22,000円 | 7,500円 | 3,300円 | 初回無料 | 月謝制 | 帰国子女（IB卒業は不明） |
| IBアカデミー | 非公開 | 月額49,800円〜 | 非公開 | 無料相談あり | 月額制 | IB卒業生（講師紹介はイニシャル） |
| TCK Workshop | 22,000円 | ランク制（最高18,000円） | 4,400円 | 非公開 | 月謝制 | 不明 |
| WAM IB | 11,000円程度 | 非公開 | 3,200円+年会費 | あり | 月謝制 | 現職教員含む |
| Kokusaba | 非公開 | 非公開 | 非公開 | 1時間無料 | 非公開 | 不明 |
| EDUBAL Academy | 非公開 | 試験対策 15,000円/90分 | 非公開 | 非公開 | コース制 | IB経験教師 |

**ポイント:** 入会金0円・管理費なし・都度払いの料金体系はIBTのみです。多くのサービスでは入会金（11,000円〜22,000円）と月額管理費（3,200円〜4,400円）が別途必要になるため、トータルコストで大きな差が出ます。

例えば、週1回60分を1年間利用した場合のおおよその年間コスト比較（DP換算）:

- **IBT:** 約45万円（8,712円 x 52回、入会金・管理費なし）
- **EDUBAL:** 約約47万円（7,500円 x 52回 + 入会金22,000円 + 管理費3,300円 x 12ヶ月）
- **IBアカデミー:** 約60万円〜（月額49,800円 x 12ヶ月。上位プランはさらに高額の可能性）

## タイプ別おすすめ

### 「まず気軽に試してみたい」なら → IBT

入会金0円・都度払いなので、**1回だけ試してみることが可能**です。60分の無料体験授業もあるので、いきなり費用がかかることはありません。「IB家庭教師ってどんな感じだろう？」と気になっている方は、まずIBTの無料体験から始めてみるのがおすすめです。

### 「大手の安心感を重視したい」なら → EDUBAL

講師数約4,000名、合格実績を大学名付きで公開。歴史と知名度があり、「大手だからこそ安心」という方に向いています。ただしIB専門ではないので、IB特化のサポートを求める場合は注意してください。

### 「月額制でガッツリ取り組みたい」なら → IBアカデミー

月額制で365日チャットサポートや月次報告書など、手厚いサポート体制が特徴です。「毎月しっかりスケジュールを組んで取り組みたい」という方に合っています。料金は月額49,800円〜と高めですが、その分サポートの範囲は広いです。

### 「IB認定校の先生に教わりたい」なら → WAM IBコース

講師にIB認定校の現職教員がいるのはWAMならではの特徴です。「実際に学校で教えている先生の視点で指導してほしい」という場合に検討する価値があります。

### 「対面でも教わりたい」なら → Kokusaba

東京近郊で対面レッスンやカフェレッスンにも対応しています。「画面越しではなく直接教えてもらいたい」という方に向いたサービスです。

## まとめ

IB家庭教師・塾を選ぶ際のポイントをまとめます。

1. **講師がIB卒業生かどうか**を必ず確認する
2. **入会金・管理費を含めたトータルコスト**で比較する
3. **お子さんのプログラム（PYP/MYP/DP）に対応しているか**を確認する
4. **無料体験授業**で実際の授業を試してみる
5. **講師のプロフィール公開状況**で信頼性を判断する

IB学習のサポートは、お子さんの学びに大きな影響を与えます。料金の安さだけでなく、講師の質や専門性、料金の透明性まで含めて、総合的に判断してください。
