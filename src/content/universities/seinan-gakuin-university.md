## 大学概要

西南学院大学は、**1916年（大正5年）**に米国南部バプテスト派の宣教師 **C.K. ドージャー** によって創立された、福岡市早良区西新にキャンパスを構えるキリスト教系の私立大学です。創立者ドージャーが遺した言葉 **"Seinan, Be True to Christ"（西南よ、キリストに忠実なれ）** を建学の精神として受け継ぎ、キリスト教を基盤とした人格教育と国際的視野の育成に力を注いできました。九州を代表する私立大学のひとつとして、「福岡から全国へ、世界へ、はばたく力を。」をスローガンに掲げています。

現在は **神学部・外国語学部・商学部・経済学部・法学部・人間科学部・国際文化学部** の7学部体制で、語学教育と国際交流プログラムに定評があります。キャンパスが位置する福岡市は「アジアの玄関口」と呼ばれ、韓国・中国・台湾・東南アジア諸国との地理的な近さから、アジア圏を意識した国際教育環境が整えられているのが特色です。IB教育が重視する国際的視野や異文化理解と、西南学院大学のキリスト教ヒューマニズムに基づく教育理念は親和性が高く、日本人IB生にとって有力な進学先のひとつとなっています。

## IB入試の仕組み

西南学院大学では、「その他の選抜」の区分の中で **国際バカロレア入試** を実施しています。国際バカロレア機構が提供する教育プログラムを履修し、高度な英語力・コミュニケーション能力・プレゼンテーション能力を備えた受験生を対象とした入試制度です（併願可）。このほか、**帰国生入試** と **総合型選抜** でもIB履修経験を活かすことができます。

**選考方法の基本構成（大学公式情報より）：**

- 書類審査（IBディプロマまたは予測スコア、成績証明書、志望理由書等）
- 小論文
- グループディスカッション
- 面接
- そのほか学部・学科が定める内容

**学部別IB要件の目安（参考・公式値ではないため要確認）：**

| 学部 | 特徴 |
|------|------|
| 国際文化学部 | 国際関係・地域研究・異文化コミュニケーションを学ぶ学部。IBの国際的視野・EEとの親和性が高い |
| 外国語学部 | 英語学科・フランス語学科等で構成。高度な語学力が求められる |
| 神学部 | キリスト教学・神学研究。建学の精神の中核を担う学部 |
| 商学部・経済学部 | 経営・会計・経済学。Economics HL・Math HLが有利 |
| 法学部 | 法律学科。社会科学系HL・小論文対策が鍵 |
| 人間科学部 | 児童教育・社会福祉・心理学。Psychology HL等が有利 |

具体的な対象学部・出願資格・IBスコア要件・試験日程は年度ごとに変更されるため、**必ず最新の入試要項を公式サイトで確認** してください。

## 出願プロセス

西南学院大学の出願は **インターネット出願** に一本化されています。国際バカロレア入試・帰国生入試を含め、公式サイトから入試要項（PDF）をダウンロードして確認のうえ、Web出願ページから手続きを進めます。

一般的な提出書類：

- 出願書類（志望理由書・自己推薦書等／学部指定のもの）
- IBディプロマ証明書または予測スコア証明書
- 高校の成績証明書・卒業（見込）証明書
- 英語資格証明（必要な場合）
- 面接・小論文・グループディスカッション等の選考への参加

**出願時期：** 学部・入試区分によって異なるため、年度ごとの最新要項で確認が必要です。総合型選抜は例年、秋（9〜11月頃）が中心となります。

## 学費と奨学金

西南学院大学の **2025年度学費は全学部共通** の体系となっています（所属学科の実験・演習等にかかる費用は別途徴収の場合あり）。

| 項目 | 入学時 | 後期 | 初年度合計 |
|------|--------|------|-----------|
| 入学金 | 200,000円 | － | 200,000円 |
| 授業料 | 375,000円 | 375,000円 | 750,000円 |
| 施設費 | 100,000円 | 100,000円 | 200,000円 |
| 教育充実費 | 20,000円 | 20,000円 | 40,000円 |

| 委託徴収金（学友会費・卒業アルバム費・保険料・同窓会費） | － | － | 34,160円 |
| **初年度納入金総額** | | | **1,224,160円** |

※ 2025年度入学生より施設費が改定されています。最新年度の金額は公式の学費明細をご確認ください。

**生活費（福岡市・参考）：** 福岡市の学生の生活コストは東京圏と比べて大幅に低く、家賃・食費込みで月7〜10万円程度が目安とされます。総生活費の安さは西南学院大学の大きな魅力のひとつです。

**主な奨学金制度（給付型）：**

- **入学試験成績優秀者奨学金** — 一般入試・共通テスト利用入試の上位合格者が対象。**初年度納付金相当額（1,190,000円）が減免**。2024年度は全学で233名が採用
- **給付奨学金（定期採用）** — 人物・学力優秀で経済的支援を必要とする学生。年額375,000円
- **成績優秀者奨学金** — 3・4年生で学業成績優秀者向け（230,000円または130,000円）
- **給付奨学金（緊急支援特別奨学金）** — 家計急変学生向け。年間授業料の半額または全額
- **C.K.ドージャー記念奨学金** — 創立者を記念したバプテスト派クリスチャン学生向け奨学金
- **河合田鶴記念奨学金** — 神学部で牧師・伝道者を志す学生向け

このほか **日本学生支援機構（JASSO）の給付型・貸与型奨学金**、民間財団、地方公共団体の奨学金も利用可能です。多子世帯支援制度の対象者は所得制限なく最大70万円の授業料減免を受けられます。

## キャンパス・学生生活

西南学院大学のキャンパスは **福岡市早良区西新6-2-92**、地下鉄空港線「西新駅」から徒歩圏内にあります。福岡タワー・百道浜・PayPayドームが近く、海に面した開放的なエリアで、周辺は学生向けの飲食店・書店・カフェが密集する住みやすい街として知られます。西新は福岡有数の学生街であり、下宿生の生活環境も整っています。

福岡は日本の主要都市の中でもアジア諸都市へのアクセスが最も良く、ソウルまで飛行機で約1.5時間、上海・台北・香港まで2〜3時間という距離感です。西南学院大学ではこの地理的優位性を活かし、アジア圏を含む海外協定校との学生交換留学・短期留学プログラムが整備されています。英語・中国語・韓国語・フランス語・スペイン語などの外国語学習環境も充実しており、語学系の課外プログラムやチャペルアワー、キリスト教ウィーク等の学院独自の行事も学生生活の特色となっています。

## 卒業後のキャリア

西南学院大学の卒業生は九州経済圏を中心に全国・海外で活躍しています。キリスト教精神に基づく人格教育を受けた卒業生は、地域社会からの信頼が厚いことでも知られます。

- **金融** — 九州の地方銀行、メガバンク、証券会社、保険会社
- **商社・メーカー** — 九州地盤の商社・食品・メーカー、全国規模の総合商社
- **航空・観光・サービス** — 航空会社地上職、ホテル、旅行業（外国語学部が強み）
- **公務員** — 国家公務員、福岡県・福岡市を中心とした地方公務員
- **マスコミ・出版** — 九州圏のテレビ局・新聞社
- **教育** — 英語教員、キリスト教系学校、国際バカロレア認定校教員
- **国際機関・NGO** — キリスト教の社会奉仕精神を活かした国際協力分野への進路
- **大学院進学** — 国内大学院、海外大学院進学

## 日本人IB生へのアドバイス

**「九州×アジアの玄関口」でIBの国際的視野を深める：** IBディプロマで培った国際的視野を、東京ではなく「アジアに最も近い日本の都市・福岡」で発展させたいIB生にとって、西南学院大学は有力な選択肢です。大学の立地そのものが、韓国・中国・東南アジアのフィールドワークやインターンシップにアクセスしやすい環境を提供しています。

**キリスト教精神と IBのCASの共鳴：** 建学の精神 "Seinan, Be True to Christ" に基づく「隣人への奉仕」「社会正義」「真理の探求」という理念は、IBのCAS（Creativity / Activity / Service）で求められる姿勢と深く重なります。志望理由書や面接で、CASでの実体験を大学の建学理念と結びつけて語れると説得力が大きく増します。

**国際バカロレア入試は「英語力＋表現力」が問われる：** 大学公式の入試説明によれば、国際バカロレア入試は「高度な英語力・コミュニケーション能力・プレゼンテーション能力」を持つ受験生を対象としています。書類・小論文・グループディスカッション・面接といった多面的な選考に備え、IBのTOK・EE・プレゼンテーション経験を整理しておくことが有効です。

**生活コストの低さは大きなメリット：** 福岡市の生活費は東京圏と比べて明らかに低く、学費と合わせた総コストを抑えられます。家計負担を抑えつつ質の高い私立大学教育を受けたいIB生にとって、実利的なメリットがあります。

**日本語での学びが中心：** 国際バカロレア入試で入学した後も、多くの学部では授業の中心言語は日本語です。英語DPで学んできたIB生は、大学入学までに学術日本語の準備を進めておくと安心です。
